DOM in JavaScript?

1. What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a webpage as a tree of objects, where each object corresponds to a part of the document, such as an element, attribute, or piece of text. JavaScript can interact with and manipulate the DOM to dynamically change the content, structure, and style of a web page.

2. How do you select an element from the DOM?

You can select elements from the DOM using various methods. The most common ones include:

  • document.getElementById('id') Selects an element by its unique ID.
  • document.querySelector('selector') - Selects the first element that matches a CSS selector.
  • document.querySelectorAll('selector') - Selects all elements that match a CSS selector and returns a NodeList.

3. What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique where you add an event listener to a parent element instead of directly to the individual child elements. When an event occurs on one of the child elements, the event bubbles up to the parent, which can handle the event for all of its children. This is useful because it reduces the number of event listeners required and makes it easier to manage dynamically added elements.

4. How do you manipulate an element's attributes and styles using the DOM?

You can manipulate an elements attributes and styles using JavaScript by accessing its properties. Some common methods include:

  • element.style.property = 'value' - Directly modifies the CSS style of an element.style.backgroundColor = 'red';.
  • element.classList.add('class') - Adds a CSS class to the element.
  • element.classList.remove('class') - Removes a CSS class from the element.
